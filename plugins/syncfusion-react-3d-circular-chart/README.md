# Syncfusion React 3D Circular Chart Plugin

This plugin bundles the Syncfusion 3D Circular Chart skill for React apps.

- Skill: skills/syncfusion-react-3d-circular-chart/SKILL.md
- Docs: skills/syncfusion-react-3d-circular-chart/references/*

What you get
- 3D Pie and Donut charts
- Data labels, legend, tooltip
- Empty point handling
- Print and export
