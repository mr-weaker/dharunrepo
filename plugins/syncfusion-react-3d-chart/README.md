# Syncfusion React 3D Chart Plugin

This plugin bundles the Syncfusion 3D Chart skill for React apps.

- Skill: skills/syncfusion-react-3d-chart/SKILL.md
- Docs: skills/syncfusion-react-3d-chart/references/*

What you get
- 3D Column/Bar/Stacked charts
- Category/Numeric/DateTime/Logarithmic axes
- Data labels, legend, tooltip
- Selection/interaction, print and export, theming, accessibility

Packages
- @syncfusion/ej2-react-charts
